Balance Calculator

Overview
This is a simple console-based Python application for managing personal finances. 
It allows the user to track income, expenses, and view transaction history. 
The program supports multiple countries and currencies.

Features
- Add income and expenses with category selection
- View current balance per country
- View full transaction history
- Switch between countries
- Supports multiple currencies (RUB, USD, CNY)
- Dual language support (English and Russian)

Installation
1. Clone the repository:
   git clone <your-repo-url>
2. Navigate to the project folder:
   cd balance-calculator
3. Run the application using Python 3:
   python main.py
   Make sure Python 3 is installed.

Usage
1. Select your language at startup (Russian or English)
2. Use the menu to:
   - Add income/expense
   - View balance
   - View transaction history
   - Select country
   - Exit
3. Follow prompts to input amounts and select categories.

Income Categories
- Work
- Gift
- Payouts

Expense Categories
- Supermarket
- Entertainment
- Bills
- Other

Countries and Currencies
- Russia – RUB
- USA – USD
- China – CNY

Калькулятор Баланса

Описание
Простое консольное приложение на Python для управления личными финансами. 
Позволяет пользователю отслеживать доходы и расходы, а также просматривать историю операций. 
Программа поддерживает несколько стран и валют.

Функции
- Добавление доходов и расходов с выбором категории
- Просмотр текущего баланса по стране
- Просмотр полной истории операций
- Смена страны
- Поддержка нескольких валют (RUB, USD, CNY)
- Двуязычная поддержка (английский и русский)

Установка
1. Склонируйте репозиторий:
   git clone <your-repo-url>
2. Перейдите в папку проекта:
   cd balance-calculator
3. Запустите приложение через Python 3:
   python main.py
   Убедитесь, что установлен Python 3.

Использование
1. Выберите язык при запуске (русский или английский)
2. Используйте меню для:
   - Добавления дохода/расхода
   - Просмотра баланса
   - Просмотра истории операций
   - Смены страны
   - Выхода
3. Следуйте подсказкам для ввода сумм и выбора категорий.

- Работа
- Подарки
- Выплаты

Категории расходов
- Супермаркет
- Развлечения
- Платежи
- Прочее

Страны и валюты
